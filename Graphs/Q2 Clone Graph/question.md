## 133: Clone Graph

**Difficulty:** Medium  
**Tags:** graph, dfs, bfs, hashmap

**Problem Reference:**

* Platform: LeetCode  
* Link: https://leetcode.com/problems/clone-graph/

**Concept to Read:**

* Graph traversal with DFS/BFS

**Issue Description:**

Create a deep copy of an undirected graph.

**Acceptance Criteria:**

* All nodes and edges are correctly cloned  
* No references to original graph remain

**Bonus:**

* Implement using both DFS and BFS
