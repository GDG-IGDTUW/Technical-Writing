## 18: Delete Duplicate Emails

**Difficulty:** Easy  
**Tags:** sql, database, delete

**Problem Reference:**

* Platform: LeetCode  
* Link: https://leetcode.com/problems/delete-duplicate-emails/description/

**Concept to Read:**

* SQL DELETE and GROUP BY

**Issue Description:**

Delete all duplicate email entries in the `Person` table, keeping only one occurrence of each unique email.

**Acceptance Criteria:**

* Remove all rows with duplicate emails except one per email  
* Ensure no unintended rows are deleted  

**Bonus:**

* Use techniques like `NOT IN`, `JOIN`, or window functions (`ROW_NUMBER`)
