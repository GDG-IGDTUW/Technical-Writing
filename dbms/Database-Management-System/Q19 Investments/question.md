## 19: Investments in 2016

**Difficulty:** Easy  
**Tags:** sql, database, filter

**Problem Reference:**

* Platform: LeetCode  
* Link: https://leetcode.com/problems/investments-in-2016/description/

**Concept to Read:**

* SQL WHERE clause and date filtering

**Issue Description:**

Report all investment records from the `Investment` table made in the year **2016**.

**Acceptance Criteria:**

* Return only records where the investment date falls in the year 2016  
* Include all columns from the table as required

**Bonus:**

* Use functions like `YEAR()` for filtering dates
