## 22: Exchange Seats

**Difficulty:** Medium  
**Tags:** sql, database, update

**Problem Reference:**

* Platform: LeetCode  
* Link: https://leetcode.com/problems/exchange-seats/description/

**Concept to Read:**

* SQL UPDATE and self-join

**Issue Description:**

Swap the seats of **two specified customers** in the `Seat` table.

**Acceptance Criteria:**

* Correctly update the seat assignments for both customers  
* Ensure no other records are affected

**Bonus:**

* Use techniques like `CASE WHEN` or joining the table with itself to perform the seat exchange
