## 25: Capital Gain/Loss

**Difficulty:** Medium
**Tags:** sql, database, arithmetic

**Problem Reference:**

* Platform: LeetCode  
* Link: https://leetcode.com/problems/capital-gainloss/description/

**Concept to Read:**

* SQL arithmetic operations and conditional logic

**Issue Description:**

Calculate the **capital gain or loss** for each transaction based on the `Transactions` table.

**Acceptance Criteria:**

* Compute the difference between selling and buying prices per transaction  
* Return the result as `gainloss` alongside `personid` and `symbol`

**Bonus:**

* Use expressions or functions like `CASE WHEN` to handle the calculation
