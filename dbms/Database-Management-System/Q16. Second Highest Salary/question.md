## 16: Second Highest Salary

**Difficulty:** Medium  
**Tags:** sql, database, ranking  

**Problem Reference:**

* Platform: LeetCode  
* Link: https://leetcode.com/problems/second-highest-salary/description/

**Concept to Read:**

* SQL aggregate functions and subqueries

**Issue Description:**

Find the second highest **distinct** salary from the Employee table. If no such salary exists, return `null`.

**Acceptance Criteria:**

* Correctly identifies the second highest distinct salary  
* Handles duplicate salary values correctly  
* Returns `null` when only one or no distinct salary exists  

**Bonus:**

* Solve using multiple approaches (subquery, LIMIT/OFFSET, window functions)
