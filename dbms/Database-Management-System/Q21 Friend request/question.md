## 21: Friend Requests II: Who Has the Most Friends

**Difficulty:** Medium  
**Tags:** sql, database, aggregation

**Problem Reference:**

* Platform: LeetCode  
* Link: https://leetcode.com/problems/friend-requests-ii-who-has-the-most-friends/description/

**Concept to Read:**

* SQL JOIN and `GROUP BY`

**Issue Description:**

Find the person (or persons) with **the most friends** based on the `Friends` table.

**Acceptance Criteria:**

* Correctly count the number of friends per person  
* Return the person ID(s) with the highest friend count

**Bonus:**

* Handle tie scenarios where multiple people have the same maximum number of friends  
* Use techniques like `JOIN`, `UNION`, and aggregation (`COUNT`)
